Overview

    This document provides guidance and specifications for the presentation of NAEP’s student-facing content. It includes the following sections, accessible via the menu above.

    • Item Development: item type usage, item language, text emphasis, language accessibility
    • Media: specifications for standard graphics, guidelines for non-standard graphics, multimedia, media accessibility
    • UX: system features and tools, student interactions, environments, UI accessibility

    This guide is a living document. It will evolve as NAEP gathers information from external research, experts in the field, and NAEP’s own studies and administrations. A dropdown menu appears at the upper left of each guide page that allows switching between the current and previous versions. Additional information about the guide’s history can be found in the version history and version scope pages of each section.
